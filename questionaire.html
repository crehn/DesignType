Q2: How would you design a List class?

There are two possibilities for constructing interfaces. Either you provide only the methods which are absolutely necessary (minimal interface) or you provide a possibly large amout of additional helper methods (humane interface). For example a List class absolutely needs methods to get/set an item at a given index and one that returns its length. Without these methods the data structure is not a real list. But one might want to add methods for returning the first() and last() element. This is not necessary as the same functionality can be achieved by using the get and length methods.
